== Description ==

As developers, we need regular doses of humour while we manage client WordPress sites. May Ron Swanson Quotes plugin give you some of that today. Source API created by jamesseanwright.

The features:

* Displays Ron Swanson quotes on your WordPress admin dashboard.
* You're welcome.

Data Notes: This plugin reaches out to the publicly available API created by jamesseanwright called ron-swanson-quotes, and [the source code can be found here](https://github.com/jamesseanwright/ron-swanson-quotes). Note that this API does not currently reside under a license of any kind.
